# React-isomorphic
## getting started
nvm use (or use node 7.x.x)

use yarn:
```
yarn
yarn start

```
use npm:
```
npm i
npm start

```
use docker:
```
docker build -t isomorphic/react:latest .
docker run -d -p 3000:3000 --name isomorphic isomorphic/react:latest

```

open on localhost:3000

## start test: 
```
npm test or yarn test

```
